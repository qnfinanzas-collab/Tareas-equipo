# TaskFlow

Gestor de tareas y equipos con planificador IA, integrado con Google Calendar.

## Arrancar

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Build para producción

```bash
npm run build
npm run preview
```

## Con Claude Code

Abre la carpeta del proyecto en Claude Code y ejecuta:

```bash
claude
```

Consulta `CLAUDE.md` para el contexto completo del proyecto.
